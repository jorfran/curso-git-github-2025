Aprendizado de Git :
Criar  branch
git checkout -b nome da branch (sai da branch main )
ir para pasta e criar um arquivo de texto estudos_git.txt
ir para notpad procurar o arquivo e inserir o texto
git status informa que o arquivo foi criado ou modificado
git add .  adiciona o arquivo
git commit -m "novo texto"  dou um nome para o commit
git log apresenta todos os commits criados para sair do log Q
git status

Para adicionar as alterações no projeto devo voltar para branch main
git checkout main
git merge estudos_git
