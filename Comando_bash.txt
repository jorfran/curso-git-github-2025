ls --> lista de arquivos
cd --> move entre diretórios
pwd --> mostra o diretório de trabalho
cat --> exibe conteúdo do arquivo
rm --> remove arquivo ou diretório
rm -rf --> remove de forma recursiva e forçada todos os diretórios 

Fluxo de Trabalho
1 Criar o diretório (pasta) mkdir(nomedapasta)
2 Iniciar o git (git init .) dentro do diretório
3 editar ou criar arquivos notpad
4 git status
5 git add nome do arquivo
6 git status
7 git commit -n "mensagem"
8 git status
9 volta para o passo 3
