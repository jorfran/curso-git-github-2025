git init .            --> cria um repositorio git na sua pasta atual
git status            --> mostra o status do trabalho
git add nome_arquivo  --> adiciona arquivo para ser comitado
git commit -m "msg"   --> faz comit do que estava para ser comitado
git log               --> mostra todos os históricos de commit

## FLUXO DE TRABALHO
1- CRIAR O DIRETÓRIO (pasta) DO PROJETO
2- INICIAR O GIT DENTRO DO DIRETÓRIO CRIADO (git init .)
3- EDITAR OU CRIAR ARQUIVOS (not pad salvar)
4- git status
5- git add nome do arquivo
6- git status
7- git commit -m "mensagem estranha"
8- git status
9- Volta passo 3
